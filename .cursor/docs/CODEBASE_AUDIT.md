## 12. Recommendations

1. **If you want to keep using the pre-built app:**  
   Continue serving from `.cursor/` and patching the bundle. No need to fix `src/` for that.

2. **If you want to develop from source:**  
   - Add `src/App.tsx` that composes all sections.  
   - Recreate sections (Hero, WhiteLabel, FAQ, etc.) in `src/components/sections/`.  
   - Import `animations.css` in `index.css`.  
   - Add `src/lib/utils.ts` and any UI components you need.  
   - Run `npm run build` and update `.cursor/index.html` to point at the new bundle.

3. **Choose one path:** Either maintain the pre-built app or migrate fully to the `src/`-based build.
